stop

stops the timer and callback timer if the setCallbackOnTimeout() and setTime() was called. After calling this function getElapsedTimeInSec() and getElapsedTimeInMS() will always return 0.

void stop();

----------------------------------------------

Remarks

if you've called setCallbackOnTimeout(), this function will block current thread until the callback timer stopped (this wait (block) will take fTimerAccuracyInSec).
