getLoadedGUIObjects

returns all loaded GUI objects.

std::vector<SGUILayer>* getLoadedGUIObjects();

----------------------------------------------

Return Value

valid pointer if successful, nullptr otherwise.

Remarks

in debug mode, some GUI objects will be created by the SProfiler for the SProfiler::showOnScreen() function, an attempt to delete such objects will result in an error (unregisterGUIObject() will return 'true'). You can check if the GUI object was created by the SProfiler using the SGUIObject::isProfilerObject() function.
