getLastFrameDrawCallCount

returns the number of the draw calls it was made to render the last frame.

bool getLastFrameDrawCallCount
(
   unsigned long long* iDrawCallCount
) const;

iDrawCallCount

pointer to your unsigned long long value which will be used to set the draw call count value.

----------------------------------------------

Return Value

false if successful, true otherwise.

Remarks

should be called on tick(), otherwise it will have an incorrect number. The function is also should be called only after calling the SApplication::run().
