makeOneCopyOfScreenPixelsToCustomBuffer

enabling this will copy screen pixels, for next frame only, to specified pPixels buffer.

void makeOneCopyOfScreenPixelsToCustomBuffer
(
   unsigned char* pPixels
);

pPixels

unsigned char array with the size equal to (width x height x 4), use getWindowSize() to get width and height.

----------------------------------------------

Remarks

you can access the pixels in onTick() function, each pixel is in 'rgba' format where each component is 'unsigned char'.
