registerCustomComputeShader

used to register custom compute shader, use returned shader functions to configure it.

SComputeShader* registerCustomComputeShader(const std::string& sUniqueShaderName);

----------------------------------------------

Return Value

valid pointer if the shader name is unique, nullptr otherwise.

Remarks

You will need to call unregisterCustomComputeShader() later when you don't need this shader anymore - this will not happen automatically, but all shaders will be unloaded in the SApplication destructor function. It's recommended to use this function in loading moments of your application (ex. loading screen) as this function may drop the framerate a little.
