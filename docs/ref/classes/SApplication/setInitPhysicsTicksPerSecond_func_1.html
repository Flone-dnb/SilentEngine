setInitPhysicsTicksPerSecond

used to set the number of physics ticks per second (i.e. the number of times the onPhysicsTick() function will be called per second) (default: 60).

bool setInitPhysicsTicksPerSecond(int iTicksPerSecond = 60);

----------------------------------------------

Return Value

false if successful, true otherwise.

Remarks

should be called before calling the SApplication::init().
