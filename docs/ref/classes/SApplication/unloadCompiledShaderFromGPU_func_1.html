unloadCompiledShaderFromGPU

used to unload and unprepare the custom shader from the GPU.

bool unloadCompiledShaderFromGPU(SShader* pShader);

----------------------------------------------

Return Value

false if successful, true otherwise.

Remarks

if any spawned object is using this shader, it will switch to the default engine shader. You can do it manually by calling the SComponent::setUseDefaultShader() function when the object is not spawned.
