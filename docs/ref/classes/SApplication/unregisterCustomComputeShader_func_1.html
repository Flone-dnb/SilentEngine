unregisterCustomComputeShader

used to unregister custom compute shader. The passed pointer will be deleted.

void unregisterCustomComputeShader(SComputeShader* pComputeShader);

----------------------------------------------

Remarks

It's recommended to use this function in loading moments of your application (ex. loading screen) as this function may drop the framerate a little.
