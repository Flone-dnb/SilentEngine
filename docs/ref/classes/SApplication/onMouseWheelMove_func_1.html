onMouseWheelMove

an overridable function, called when the user moves the mouse wheel.

virtual void onMouseWheelMove
(
   bool bUp,
   int iMouseXPos,
   int iMouseYPos
);

bUp

true if the mouse wheel was moved up (forward, away from the user), false otherwise.

iMouseXPos

mouse X position in pixels.

iMouseYPos

mouse Y position in pixels.
