getCursor3DPosAndDir

used to retrieve the cursor position in 3D space (camera position) and it's direction from the camera (eyes).

bool getCursor3DPosAndDir(SVector& vPos, SVector& vDir);

----------------------------------------------

Return Value

false if successful, true otherwise.

Remarks

this function will return an error if it was called when the cursor is hidden, or if run() was not called before.
