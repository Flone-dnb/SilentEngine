setUseDefaultShader

used to switch the custom used shader to a default one.

bool setUseDefaultShader
(
   bool bForceUseDefaultEvenIfSpawned = false
);

bForceUseDefaultEvenIfSpawned

forces the engine to change the shader even if this mesh is spawned (causes small fps drop).

----------------------------------------------

Return Value

false if successful, true if this mesh is spawned and the 'bForceUseDefaultEvenIfSpawned' is false.
