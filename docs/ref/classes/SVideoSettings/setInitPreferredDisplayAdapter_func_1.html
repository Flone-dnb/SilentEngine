setInitPreferredDisplayAdapter

used to set the preferred display adapter (i.e. "video card" on your PC). The list of all supported display adapters may be retrieved through SVideoSettings::getSupportedDisplayAdapters() function.

bool setInitPreferredDisplayAdapter
(
   std::wstring sPreferredDisplayAdapter
);

sPreferredDisplayAdapter

the name of the preferred display adapter.

----------------------------------------------

Return Value

false if successful, true otherwise.

Remarks

should be called before calling the SApplication::init().
