setInitPreferredOutputAdapter

used to set the preferred output adapter (i.e. monitor on your PC). The list of all supported output adapters may be retrieved through SVideoSettings::getOutputDisplaysOfCurrentDisplayAdapter() function.

bool setInitPreferredOutputAdapter
(
   std::wstring sPreferredOutputAdapter
);

sPreferredOutputAdapter

the name of the preferred output adapter.

----------------------------------------------

Return Value

false if successful, true otherwise.

Remarks

should be called before calling the SApplication::init().
