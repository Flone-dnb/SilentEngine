setShadowMappingBias

used to change the depth bias for shadow mapping.

void setShadowMappingBias(int iShadowMappingBias);

----------------------------------------------

Remarks

the depth bias is a scene dependent value, for some scenes it may be necessary to change this value to reduce the so called 'shadow acne' effect. Note that this effect is caused by the limited resolution of the shadow map, if you are using small resolution for your shadow maps, it may be better to increase the resolution, because big depth bias values cause the so called 'peter-panning' artifact.
