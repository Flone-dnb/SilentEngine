SGUILayout

creates a layout with the specified width and height (in normalized range: [0, 1]).

SGUILayout
(
   const std::string& sObjectName,
   float fWidth,
   float fHeight,
   SLayoutType layoutType,
   bool bExpandItems
);

sObjectName

name of this object.

fWidth

width to keep of this layout (in normalized range: [0, 1]).

fHeight

height to keep of this layout (in normalized range: [0, 1]).

layoutType

type of this layout.

bExpandItems

whether to stretch layout items so they will fill the whole layout space or don't stretch them to keep their original size.

----------------------------------------------

Remarks

if this layout will be used in another layout the width and the height will be ignored.
