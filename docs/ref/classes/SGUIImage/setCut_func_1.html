setCut

use to specify the rectangle (in order: left, top, right, bottom corner) for drawing just part of a GUI object (in normalized range: [0, 1]).

void setCut(const SVector& vSourceRect);

----------------------------------------------

Remarks

for example, passing the (x: 0.0f, y: 0.0f, z: 0.5f, w: 0.5f) will cut the GUI object to render only left-top corner of the object (all relative to top-left corner, not the origin).
