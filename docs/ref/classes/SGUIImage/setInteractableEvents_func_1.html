setInteractableEvents

use to specify if the user can click on this image. Interactable images allow binding to such events as onHover and onPressed.

void setInteractableEvents
(
   std::function<void(SGUIImage*)> noFocus,
   std::function<void(SGUIImage*)> onHover,
   std::function<void(SGUIImage*)> onPressed
);

noFocus

called on cursor move event when it's (cursor) outside of the image bounds.

onHover

called on cursor move event when it's (cursor) in the bounds of this image.

onPressed

called when the cursor is in the bounds of this image and the user pressed left mouse button.
