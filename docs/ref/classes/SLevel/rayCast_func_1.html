rayCast

used to emit a ray that hits the components with collision.

void rayCast
(
   SVector& vRayStartPos,
   SVector& vRayStopPos,
   std::vector<SRayCastHit>& vHitResult,
   const std::vector<SComponent*>& vIgnoreList = std::vector<SComponent*>()
);

vRayStartPos

start position of the ray.

vRayStopPos

stop (end) position of the ray.

vHitResult

objects that were hit by the ray, use SComponent::getComponentType() to cast to correct type.

vIgnoreList

(optional) list of component that will be ignored in this ray cast.
