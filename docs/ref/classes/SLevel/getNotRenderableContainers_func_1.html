getNotRenderableContainers

returns all non-renderable containers (containers that don't have components with geometry in it, for example, STargetComponent) in level.

(
   std::vector<SContainer*>*& pvNotRenderableContainers
);

pvNotRenderableContainers

the pointer to your std::vector that will be filled with pointers to the non-renderable containers in level. You should not spawn/despawn containers while working with the returned vector as the vector length will change. Use synchronization techniques.
