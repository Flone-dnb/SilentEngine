startShaderExecution

starts to execute shader on every draw call until stopShaderExecution() is called.

bool startShaderExecution
(
   unsigned int iThreadGroupCountX,
   unsigned int iThreadGroupCountY,
   unsigned int iThreadGroupCountZ
);

iThreadGroupCountX

defines the amount of thread groups that will be dispatched by X-axis. Must be at least 1.

iThreadGroupCountY

defines the amount of thread groups that will be dispatched by Y-axis. Must be at least 1.

iThreadGroupCountZ

defines the amount of thread groups that will be dispatched by Z-axis. Must be at least 1.

----------------------------------------------

Return Value

returns true if the compute shader (hlsl) was not specified (see SComputeShader::compileShader()) or no shader data is added.
