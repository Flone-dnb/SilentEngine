getIndices16

returns the indices in the format of std::uint16_t. This means that the highest value of the index is 65535. Use SMeshData::hasIndicesMoreThan16Bits() to determine if the mesh data has indices with value more than the uint16_t max value, and use getIndices32() if it does.
