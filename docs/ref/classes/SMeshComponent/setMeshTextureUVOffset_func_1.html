setMeshTextureUVOffset

used to set the UV offset to the mesh texture. Only affects how the textures will look for THIS mesh. This setting will affect material UV settings only for this mesh.

bool setMeshTextureUVOffset(const SVector& vMeshTexUVOffset);

----------------------------------------------

Return Value

true if the UVs are not in [0, 1] range, false otherwise.

Remarks

should be in [0, 1] range.
