setEnableTransparency

determines if the material on this component should consider the alpha channel of the diffuse texture or a custom value from SMaterial::setCustomTransparency().

bool setEnableTransparency(bool bEnable);

----------------------------------------------

Return Value

false if successful, true if this component is spawned.

Remarks

this value can only be changed before this component is spawned. Transparency is disabled by default.
