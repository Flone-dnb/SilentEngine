removeChildComponent

removes a child component.

bool removeChildComponent(SComponent* pComponent);

----------------------------------------------

Return Value

false if successful, true otherwise.

Remarks

1) if the container, containing these components has already been spawned then you cannot add/remove components; 2) if this component is not a part of any container then you can't add a child component to this component; 3) if the child component does not have a unique name (within the boundaries of this container) then this function will fail.
