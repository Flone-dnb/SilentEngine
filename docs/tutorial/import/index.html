Import


In order to import an object use .OBJ format.

When exporting from Blender pick these options:

  • [Include - Limit to] Selection only (to export only selected object)
  • [Include - Objects as] OBJ Objects
  • [Transform - Forward] Y Forward
  • [Transform - Up] Z Up
  • [Geometry] Apply Modifiers
  • [Geometry] Write normals
  • [Geometry] Include UVs
  • [Geometry] Triangulate faces

Use SFormatOBJImporter::importMeshDataFromFile() to import the file in the engine.
