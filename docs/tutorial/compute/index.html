Compute Shaders


In order to stop a compute shader, you need to use the onTick() function.

First, call SComputeShader::stopShaderExecution(), then on the next tick use SApplication::unregisterCustomComputeShader() and nullptr your compute shader pointer. No logic is needed on the next tick.

Frustum culling will be disabled for mesh if the SMeshComponent's mesh data is used in the compute shader (via SMeshComponent::getMeshDataAsComputeResource()).
