Visual Effects


Use SApplication::getGlobalVisualSettings to control things like ambient light, distant fog and etc.

Use SCamera::setCameraEffects() to control things like gamma, saturation, blur and etc.

The SMeshComponent::setDrawAsLines() only works if the mesh is using default shader and if the mesh material is not using transparency.

In order to have a skybox you need:

  1. Create a sphere with a radius of far clip plane (get from SCamera).
  2. Load the cubemap texture (a texture that contains 6 textures in it) for skybox using SApplication::loadTextureFromDiskToGPU() (set 'bIsCubeMap' to true).
  3. Register the shader /shaders/skybox.hlsl using SApplication::compileCustomShader() and in 'customProps' pass the cubemap texture handle as 'skyboxTexture'.
  4. Set the sphere to use this custom shader (using SComponent::setUseCustomShader()).
  5. The skybox shader will automatically move the sphere to camera position on each frame.

For Instancing use SMeshComponent's constructor with 'bUseInstancing' to 'true'. You can then use functions like: SMeshComponent::addInstance(), SMeshComponent::updateInstanceData(), SMeshComponent::clearAllInstances(). Note than you need to have a custom shader for instanced mesh, you can find an example shader that you can use in /shaders/optional/. Mainly, instancing code is applied in the vertex shader (the VS() function in the .HLSL file), other parts of the shader is equal to the basic.hlsl.
