Camera attachment


In order to attach a camera to something, like a head of your character, and have it in the component hierarchy, you need to use STargetComponent.

Add STargetComponent to the component hierarchy. It will be used as a placeholder for camera location.

Use SComponent::setBindOnParentLocationRotationScaleChangedCallback() on this STargetComponent and set camera location in this function based on the STargetComponent location in world.

This way you can have a first person camera, for example.

In order to have a third person camera, you can use SCamera::setCameraMode() with CM_FIXED.
